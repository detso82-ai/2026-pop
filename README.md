# 👥 쌍둥이 동네 찾기 (Twin Neighborhood Finder)

행정안전부 연령별 인구현황 데이터를 이용하여 **선택한 지역과 인구구조가 가장 비슷한 지역**을 찾아주는 Streamlit 웹앱입니다.

**Demo:** Streamlit Cloud 배포 예정

---

# 주요 기능

## 1. 지역 검색

- 전국 모든 행정구역 검색
- 시·도
- 시·군·구
- 읍·면·동
- 검색 + 선택 방식 지원

---

## 2. 연령별 인구구조 시각화

선택한 지역의

- 전체 인구
- 남성
- 여성

인구구조를 Plotly 인터랙티브 그래프로 제공합니다.

지원 기능

- 확대
- 축 이동
- PNG 저장
- Hover 정보
- Range Slider

---

## 3. 인구구조가 비슷한 지역 찾기

선택한 지역과 전국 모든 지역을 비교하여

**Top 5 유사 지역**을 제공합니다.

비교 기준

- 연령별 인구 "수"가 아닌
- 연령별 인구 "비율"

을 이용합니다.

즉,

> 총인구가 달라도 연령구조의 모양이 비슷하면 높은 점수를 받습니다.

---

## 4. 유사도 계산

각 지역을

```
0세
1세
2세
...
100세 이상
```

총 101차원의 인구비율 벡터로 변환합니다.

그 후

**Cosine Similarity**

를 이용하여 유사도를 계산합니다.

유사도는

- 100점에 가까울수록 매우 유사
- 0점에 가까울수록 매우 다름

입니다.

---

## 5. 유사 지역 비교

선택 지역과

Top5 지역의

연령별 인구구조를 하나의 Plotly 그래프에서 비교할 수 있습니다.

---

## 6. 결과 다운로드

- CSV 다운로드
- Plotly PNG 저장

지원
